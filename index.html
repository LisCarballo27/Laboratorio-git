Descargar Git

¿Qué es Git?

Git es un sistema de control de versiones que permite guardar el historial de cambios y trabajar en equipo.

Ventajas

  • Trabajo en equipo
  • Control de versiones
  • Uso de ramas
  • Recuperación de cambios

Comandos básicos

  • git init
  • git add .
  • git commit -m "mensaje"
  • git status

Repositorios remotos

  • git remote add origin URL
  • git push -u origin main
  • git pull

Conceptos clave

  • Repositorio: donde se guarda el proyecto
  • Commit: guardar cambios
  • Branch: rama de trabajo
  • Merge: unir cambios
